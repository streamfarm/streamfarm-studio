# StreamFarm Studio <https://streamfarm.ru>

## What is StreamFarm Studio?

**StreamFarm Studio** is a desktop application for capturing, compositing, encoding, recording, and streaming video content directly from your PC.

The application is **a customized fork of OBS Studio**, enhanced and optimized for deep integration with the StreamFarm platform. While retaining the power and flexibility of OBS, StreamFarm Studio includes additional features, streamlined workflows, and direct cloud integration tailored for creators using the StreamFarm ecosystem.

With built-in connection to the **StreamFarm online service**, creators can manage all aspects of their broadcasts â€” from setting up scenes and sources to launching streams in just a few clicks.

## Key Features

- Capture video from screen, windows, cameras, and other media sources  
- Scene and layer-based video composition  
- Advanced audio mixing  
- Local recording with flexible encoding options  
- One-click streaming through StreamFarm Cloud  
- Account connection and cloud profile syncing  
- Planned integrations: chat overlay, donation tools, real-time analytics  
- Retains OBS performance and plugin ecosystem where compatible
